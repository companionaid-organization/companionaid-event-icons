# companionaid-event-icons

##№ Что здесь есть, и для чего нужен репозиторий? 

**В состав репозитория входит три файла**

1. event-config.json
   В нем описано состояние каждого события: event1 и event2.
   Если _"isVisible": true_ событие, для которого оно обозначено, должно отображаться, если _"isVisible": false_ - не должно.
   Структура файла продублирована ниже. По описанной ниже настройке можно понять, что event1 должно отображаться, а event2 - нет.
   
   ```
   {
     "event1": {
       "isVisible": true
     },
     "event2": {
       "isVisible": false
     }
   }
   ```
2. event1.svg
   Этот файл содержит иконку в формате svg для события1.
   
3. event2.svg
   Этот файл содержит иконку в формате svg для события2.

**Эти иконки используются на странице карты безопасности:**

1) Для пинов, связанных с событиями.
2) Для фильтров по соответствующим событиям.

##№ При каком условии идет отображение пинов и фильтров на сайте?

Пины и соответствующие фильтры отображаются, если **ОДНОВРЕМЕННО** выполняются **ОБА** условия (для примера возьмем event1):
1. В event-config.json для event1 поле в **"isVisible"** стоит переменная **true**.
2. В репозитории **существует** файл иконки для этого события (event1.svg) **И** этот **файл не пустой**, а содержит изображение в формате svg.

##№ Как скрыть или показать пины и фильтры на сайте?

Чтобы скрыть событие с сайта (его пины и фильтр) достаточно заменить переменную **"isVisible"** внутри event-config.json соответствующего собятия на **false**. При этом файл иконки можно не менять, не удалять.

##№ Как изменить иконку для события?

Чтобы изменить/обновить иконку для одного из событий, нужно выполнить два шага:

1. Заменить иконку. Для этого есть **два пути**:
   
   - **первый - аналогичный изменению кода в файле конфига**
     1) Вам нужно экспортировать нужную иконку из фигмы в формате .svg
     
       <img width="672" height="487" alt="image" src="https://github.com/user-attachments/assets/b0ce2d07-d7a8-454d-84d7-f78a9fbe1f77" />
     
     2) Открыть этот файл в блокноте и скопировать содержимое:

      <img width="1141" height="634" alt="image" src="https://github.com/user-attachments/assets/3e688b27-218a-4f41-b621-ada45cd6701c" />

     3) Аналогично изменению кода в файле конфига, заменить код в файле изображения и сохранть изменения, кликнув на зеленую кнопку в верхнем правом углу, подтвердив действие кликом на аналогичную кнопку в поп-апе. Единственное различие в том, что при открытии файла .svg вы сначала увидите его предпросмотр (саму иконку), чтобы увидеть код, нужно выбрать вкладку _Code_, которая откроется автоматически при клике на иконку карандаша (Edit this file).
